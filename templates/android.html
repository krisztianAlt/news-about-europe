    In the mobile version, you can select from not four, but six news agencies. Countries are represented by their flags instead of map.

    You can reorder the list of articles by title, author and date, in ascending and descending order too. You can mark articles as your favorite.

    Favorite articles are saved, so you can list them later.

    Mobile version works with your own API key, so you should register on News API for the key. Select the "Developer" option, it's free. You will get the key in e-mail.

    You have two options to install the app.

    1. Visit News About Europe site in the browser of your mobile device, and touch the Download button below. Save the APK file into your device's Downloads folder. Then you should start the Files app, touch Downloads, then select and install news-about-europe-for-android.apk.
